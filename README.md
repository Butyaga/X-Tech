# Тестовое задане одного из потенциальных работодателей.

## Задание 1
Создать приложение в паттерне MVVM, которое содержит одно окно (Window) с элементами:
- Кнопка «Сканировать». Осуществляет сканирование доступных WiFi-сетей. Выводит полученный результат в таблицу
- Кнопка «Сохранить». Сохраняет полученный список сетей в базу данных
- Текстовое поле (TextBox) или метка (Label), в которую выводится название сети с наилучшим сигналом
- Таблица (DataGrid) для вывода полученных Wifi-сетей. Должна содержать столбцы Имя(SSID) сети, уровень сигнала сети.

При нажатии кнопки «Cканировать» полученный список сетей должен отображаться в таблицу, а название сети с максимальным уровнем сигнала присваиваться содержимому метки или тексту текстового поля. При нажатии кнопки «Сохранить» список полученных сетей должен сохраняться в базу данных.
